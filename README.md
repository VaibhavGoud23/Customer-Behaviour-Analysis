# Customer-Behaviour-Analysis
This project is an end-to-end data analytics workflow where raw data is transformed into meaningful insights. I worked on loading and analyzing the dataset using Python, cleaning and preparing it for analysis, running SQL queries in MySQL, and finally building an interactive Power BI dashboard.

 -- Dataset
Source: ( customer_shopping_behaviour)
Format: CSV 
The dataset includes information related to customer behavior/business performance, which was used to identify patterns and trends.

🛠️ Tools & Technologies
Python (Pandas, NumPy)
Matplotlib & Seaborn (for visualization)
MySQL (for data storage and querying)
SQLAlchemy (for connecting Python with MySQL)
Power BI (for dashboard creation)

 -- Project Steps

1. Data Loading
I started by importing the dataset into Python using Pandas and explored its structure, columns, and data types.

2. Exploratory Data Analysis (EDA)
Performed basic analysis to understand the data. Checked distributions, trends, and identified missing values and outliers using visualizations.

3. Data Cleaning
Cleaned the dataset by handling missing values (using median/mode), removing duplicates, and fixing inconsistent formats.

4. SQL Analysis (MySQL)
Loaded the cleaned dataset into MySQL and ran SQL queries to extract useful insights such as totals, averages, and category-wise analysis.

5. Dashboard Creation (Power BI)
Connected Power BI to the database and built an interactive dashboard with key metrics, charts, and filters for better understanding.

6. Reporting & Presentation
Finally, created a report summarizing the insights and built a presentation using Gamma to communicate the results effectively.

-- Dashboard

The dashboard provides a clear and interactive view of the data, including:

Key performance indicators (KPIs)
Visual charts for trends and comparisons
Filters for dynamic analysis
